# Car-Service-API
Simple API for realizing some car services with users as sellers
## Installation
```
git clone git@github.com:darten360/taxi-service-darten.git
cd taxi-service-darten
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python app.py
```
